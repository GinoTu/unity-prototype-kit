# Demo Scene

這個 Sample 展示 Gino Prototype Kit 的最小可跑組合。

## 安裝

Package Manager → Gino Prototype Kit → Samples → **Install Demo**

## 包含內容

- `DemoScene.unity`：一個可以直接 Play 的示範場景
  - TopDown Player（WASD 移動）
  - HUD（血條 + 分數 + 計時器）
  - GameManager（按 Escape 切換暫停）

## 如何從這裡開始做你的 Prototype

1. 把 `DemoScene` 複製到你的 Assets 資料夾
2. 把不需要的元件刪掉
3. 在 Player 上換掉你自己的美術素材
4. 開始加你的遊戲邏輯
